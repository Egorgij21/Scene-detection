# Scene-detection

how to start the code in terminal:

0) git clone https://github.com/Egorgij21/Scene-detection <path/to/repo>
1) cd <path/to/repo>
2) mkdir build
3) cd build
4) cmake ..
5) make
6) cd ..
7) cd bin
8) ./detect <path/to/video> <path/to/folder_for_cutted_videos> <(int) threshold> <Filter name> <(int) kernel Size>
